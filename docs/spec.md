# Webサイト要件定義・仕様書

> **プロジェクトコード:** Data & Canvas  
> **作成日:** 2026-03-07  
> **目的:** Google Antigravity でのバイブコーディング用リファレンスドキュメント

---

## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| **サイト名（仮）** | Data & Canvas |
| **目的** | 個人で開発したアプリケーション・分析ツールのポートフォリオ展示と、日々の学習・ライフスタイル（作業カフェ記録など）の発信拠点 |
| **ターゲット** | テクノロジー・自己研鑽・投資に興味があり、洗練されたモダンなデザインを好むユーザー |
| **運用方針** | 深い専門知識がなくても、直感的にコンテンツの追加・更新ができるシンプルな管理構造 |

---

## 2. システム・環境要件

- **開発プラットフォーム:** Google Antigravity
- **対応デバイス:** フルレスポンシブ対応（PC・タブレット・スマートフォン）
- **パフォーマンス:** 画像の最適化・不要なアニメーション削減により、快適な表示速度を確保

---

## 3. サイトマップとコンテンツ構成

### 3-1. Home（トップページ）

- **Hero セクション:** キャッチコピーと洗練された背景画像
- **Projects ピックアップ:** 最新2件をカード型で表示
- **Journal 最新記事一覧:** 記事サムネイル・タイトル・日付のプレビュー
- **フッター:** ナビゲーションリンク、SNS リンク

### 3-2. Projects（制作物ギャラリー）

- カード型 UI による一覧表示
- 各カードに以下を設置:
  - タイトル
  - 概要説明
  - 使用技術タグ（例: `Python`, `Antigravity`）
  - GitHub リンク / アプリリンク
- 例示コンテンツ:
  - 株価分析ダッシュボード
  - 英語学習サポートアプリ

### 3-3. Journal（ブログ・記録）

- **記事一覧ページ:** サムネイル・タイトル・日付・タグを表示
- **タグフィルタ機能:**
  - `#Development`
  - `#English`
  - `#Cafe`
  - `#Mindfulness`
- **CMS 機能:** テキストと画像の追加のみで記事生成が完結

### 3-4. About（プロフィール）

- シンプルな自己紹介テキスト
- スキルセット・興味分野のリスト

### 3-5. Contact（問い合わせ）

- 入力項目: お名前・メールアドレス・メッセージ（3項目のみ）
- 送信完了メッセージの表示

---

## 4. デザイン仕様

### 4-1. カラースキーム

| 用途 | カラー名 | カラーコード |
|---|---|---|
| **ベースカラー** | オフホワイト | `#F9F9F9` |
| **メインカラー** | ダークグレー | `#333333` |
| **アクセントカラー** | コーヒーブラウン | `#8B5A2B` |

- ベースカラー: 余白を活かすクリーンな背景
- メインカラー: テキストや主要 UI 要素
- アクセントカラー: ボタン・リンクホバー時など、落ち着いたアクセント

### 4-2. タイポグラフィ

- 和文・欧文ともにモダンなサンセリフ体（ゴシック体）を使用
- 視認性・可読性を最優先

### 4-3. UI / UX の方向性

- **余白重視:** ホワイトスペースを十分に確保し、要素を詰め込まない
- **マイクロインタラクション:** ホバー時の軽い浮き上がり・フェードインなど、品のある演出に留める
- **禁止事項:** 複雑なパララックスや過度なエフェクトは採用しない

---

## 5. 開発時の制約・ガイドライン

### 5-1. コードのシンプルさ（最優先）

- 複雑なフロントエンド・バックエンドの分離は採用しない
- オーバースペックなアーキテクチャは不要
- **保守性を最優先**とし、誰でも読める・直せるコードを書く

### 5-2. AI プロンプト運用方針

- 専門用語の羅列による過剰な演出は行わない
- 純粋な要件定義の実現のみにフォーカス
- フラットで平易なコミュニケーションを維持
